# Скрипта за поздравување

Напишете Bash скрипта **`greet.sh`** што поздравува корисник по име.

## Барања

1. Кога се повикува со точно **еден аргумент** (името), испечатете на **stdout**:

   ```
   Hello, <name>!
   ```

   и излезете со статус **0**.

2. Кога се повикува **без аргументи**, испечатете порака за употреба на **stderr**:

   ```
   usage: greet.sh <name>
   ```

   и излезете со статус **1**. (Ништо не треба да се печати на stdout.)

3. Скриптата мора да помине `shellcheck` без предупредувања.

## Примери

```console
$ ./greet.sh World
Hello, World!
$ echo $?
0

$ ./greet.sh
usage: greet.sh <name>     # this line goes to stderr
$ echo $?
1
```

> **Совет:** `"$1"` е првиот аргумент; `$#` е бројот на аргументи. Испратете текст на
> stderr со `>&2`.
