# Лабораторија за скрипти — Копирање на CSV датотеки и извештај со статистика

Напишете shell скрипта што ги копира сите датотеки од изворен директориум што ги задоволуваат **сите**
од овие услови:

- имаат **`.csv`** наставка,
- нивното **име на датотека содржи барем една цифра**,
- **другите** (others) имаат **право на читање** на датотеката.

## Барања

- Изворниот директориум е даден како **аргумент од командната линија**.
- Создадете директориум **`copiedFiles`** во тековниот директориум и копирајте ги соодветните
  датотеки во него.
- Исто така создадете **`statistics.txt`** со **точно** овој формат:

  ```
  copiedFiles: X
  totalSize: Y
  totalNumberOfLines: Z
  largestFile: A
  longestName: B
  ```

  каде што `X` = бројот на копирани датотеки, `Y` = нивната вкупна големина во бајти,
  `Z` = нивниот вкупен број на линии, `A` = големината (бајти) на најголемата копирана
  датотека, а `B` = името на таа најголема датотека.

## Пример

```console
$ mkdir source && cd source
$ echo "a" > file1.csv; echo "b" > file2.csv; echo "c" > test.csv
$ chmod o+r file1.csv file2.csv; chmod o-r test.csv
$ cd ..
$ ./script.sh source
```

`statistics.txt`:

```
copiedFiles: 2
totalSize: 4
totalNumberOfLines: 2
largestFile: 2
longestName: file1.csv
```

(`test.csv` се прескокнува — нема цифра во неговото име.)

> **Совет:** совпаднете цифра во името со `case` glob `*[0-9]*`; прочитајте го битот за дозвола на
> „другите" (others) од `stat -c '%A'` (8-миот знак); големини преку `stat -c%s`.
