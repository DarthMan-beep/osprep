# Лабораторија за скрипти — TCP конекции по локална IP адреса

Напишете shell скрипта што, за податоци прочитани од **стандарден влез** (во форматот на
`netstat -an`), го печати бројот на **TCP** конекции за секоја **локална адреса**
(IP без портата).

Секоја излезна линија го користи форматот `"%-20s %s\n"`: прво локалната IP адреса (без порта), потоа
бројот на TCP конекции за неа.

Подредете го излезот:

- **опаѓачки** по бројот на конекции;
- за еднакви броеви, **растечки** по IP адресата.

## Правила

- Земете ги предвид само TCP редовите (`tcp` и `tcp6`); игнорирајте ги линиите со заглавие.
- Отстранете ја портата: `0.0.0.0:80` → `0.0.0.0`, и IPv6 wildcard `:::80` → `:::`.
- Користете `awk`, `sort`, `uniq` (или еквивалентно решение само со `awk`). **Без привремени датотеки.**
- Читајте **само** од stdin.

## Пример

Влез:

```
Active Internet connections (servers and established)
Proto Recv-Q Send-Q Local Address  Foreign Address  State
tcp   0 0 192.168.1.10:8080  0.0.0.0:*  LISTEN
tcp   0 0 10.0.0.5:3306      0.0.0.0:*  LISTEN
tcp   0 0 10.0.0.5:443       0.0.0.0:*  LISTEN
tcp6  0 0 :::443             :::*       LISTEN
tcp   0 0 192.168.1.10:22    0.0.0.0:*  LISTEN
```

Излез:

```
10.0.0.5             3
192.168.1.10         2
:::                  1
```

> **Совет:** локалната адреса е колона 4. Отстранете ги завршните цифри, потоа отстранете една
> завршна две точки **само** кога следи по знак што не е две точки (така што `:::` се зачувува).
