# Подигни ја услугата

Build контекстот содржи **`server.py`** — мала HTTP услуга. Напиши **`Dockerfile`** што ја
извршува така што, кога контейнерот ќе се подигне, ова барање ќе успее:

```
GET /health  ->  HTTP 200, телото содржи "healthy"
```

## Што мора сам да откриеш

- Оценувачот ја објавува портата на контейнерот наместо тебе и потоа го прави барањето — но
  **ти мора да го стартуваш серверот кога ќе се изврши контейнерот** (базната слика нема
  сама да го извршува `server.py`).
- **На која порта** слуша? Не погодувај — отвори го `server.py` и прочитај. Твојата слика
  треба да ја изложи таа порта.

> Не се потребни дополнителни пакети (`server.py` користи само стандардната библиотека на
> Python), па `python:3.12-slim` база е доволна. Работата е: внеси го кодот и направи
> контейнерот да го стартува.
