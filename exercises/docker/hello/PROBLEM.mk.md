# Твојот прв Dockerfile

Во build контекстот е дадена мала Python програма **`main.py`**:

```python
print("Hello from Docker!")
```

Напиши **`Dockerfile`** што ја пакува и ја извршува. Кога сликата ќе се изгради и од неа
ќе се стартува контејнер, контејнерот мора да испечати:

```
Hello from Docker!
```

## Барања

Твојот Dockerfile треба да:

1. почне **`FROM`** Python базна слика (на пр. `python:3.12-slim`),
2. постави работен директориум со **`WORKDIR`**,
3. **`COPY`**-ра `main.py` во сликата,
4. ја извршува при стартување со **`CMD`**.

> **Совет:** четирите инструкции што ти требаат се `FROM`, `WORKDIR`, `COPY` и `CMD`.
> `CMD` прима JSON низа, на пр. `CMD ["python", "main.py"]`. Секоја инструкција станува
> слој; `COPY main.py .` копира во тековниот `WORKDIR`.
