# Испорачај ја целата апликација

Build контекстот содржи мала Python програма. Нејзиното извршување треба да печати:

```
Total inventory value: 12000
```

Напиши **`Dockerfile`** што ја пакува и ја извршува со Python.

## Замката

Програмата **не е една датотека** — `main.py` зависи од друг модул во истата папка. Ако
сликата содржи само `main.py`, Python ќе падне при извршување со нешто како
`ModuleNotFoundError: No module named 'inventory'`. Обезбеди сликата да содржи сè што ѝ
треба на програмата.

> Базната слика `python:3.12-slim` веќе има Python; тебе ти треба кодот (целиот) на место
> и команда што го извршува `main.py`.
