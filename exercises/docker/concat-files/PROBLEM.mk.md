# Копирај две датотеки и спој ги

Во build контекстот се дадени две текстуални датотеки: **`intro.txt`** и **`outro.txt`**
(можеш да ги прочиташ во панелот *Дадени датотеки* подолу).

Напиши **`Dockerfile`** што ги копира **двете** датотеки во сликата и, кога контейнерот се
извршува, ги печати **една по друга** така што излезот гласи:

```
Welcome to the inventory system
```

Која било мала Linux база работи (на пр. `debian:12-slim`).

> **Совети:** едно `COPY` може да прими повеќе извори пред дестинацијата, на пр.
> `COPY intro.txt outro.txt ./` ги копира двете во тековниот `WORKDIR`. За да печатиш
> датотеки по редослед, `cat` прима повеќе имиња: `cat intro.txt outro.txt`. (Ако
> заборавиш да копираш една, `cat` ќе даде грешка при извршување бидејќи датотеката ја нема
> во сликата.)
