# Лаб 1 — Изброј процеси по корисник

Читајќи **податоци во стил на `ps aux` од стандардниот влез**, испечати го бројот на процеси
по корисник.

- Секоја линија од излезот мора да го користи форматот `"%-20s %s\n"` — првото поле е
  **корисникот**, второто е бројот на процеси на тој корисник.
- Подреди го излезот во **опаѓачки редослед според бројот на процеси**.
- Првата линија од излезот на `ps aux` е **заглавие** и мора да се игнорира.

## Пример

Влез на stdin:

```
USER  PID ... COMMAND
root    1 ... init
alice   2 ... bash
alice   3 ... vim
bob     4 ... top
alice   5 ... less
bob     6 ... cat
```

Излез:

```
alice                3
bob                  2
root                 1
```

> **Совет:** изгради цевковод (pipeline) — извлечи ја колоната со корисникот (прескокнувајќи го заглавието), потоа
> `sort | uniq -c | sort -nr`, па преформатирај со `awk` и `printf`.
