# Lab 1 — Create directories and a file

Write the command(s) that:

1. Create a directory named **`directory1`**.
2. Create a directory named **`directory2`**.
3. Inside `directory1`, create an empty file named **`file123`**.

## Expected result

```console
$ ls directory1
file123
$ ls directory2
```

> **Tip:** `mkdir` makes a directory; `touch` creates an empty file. You can chain
> commands with `&&` so the next runs only if the previous succeeded.
