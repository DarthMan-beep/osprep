# Лаб 1 — Креирај директориуми и датотека

Напиши ја командата(ите) што:

1. Креира директориум со име **`directory1`**.
2. Креира директориум со име **`directory2`**.
3. Во `directory1`, креира празна датотека со име **`file123`**.

## Очекуван резултат

```console
$ ls directory1
file123
$ ls directory2
```

> **Совет:** `mkdir` креира директориум; `touch` креира празна датотека. Можеш да поврзеш
> команди со `&&` така што следната се извршува само ако претходната успеала.
