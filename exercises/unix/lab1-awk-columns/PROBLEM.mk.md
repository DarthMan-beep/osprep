# Лаб 1 — Печатење на избрани колони

Секоја линија од **`OS1.txt`** и **`OS2.txt`** ги има полињата:

```
<index> <exam-date> <quiz-status> <points>
```

Напишете ја командата што за **секоја линија од двете датотеки заедно** ги печати
**индексот** и **статусот на квизот** (полиња 1 и 3), одделени со празно место.

## Пример

`OS1.txt`:
```
21001 14.03.2024 done 80
22002 15.03.2024 in_progress 45
```
`OS2.txt`:
```
21003 14.03.2024 done 30
```
Излез:
```
21001 done
22002 in_progress
21003 done
```

> **Совет:** `awk '{ print $1, $3 }'` може да прифати повеќе датотеки одеднаш.
