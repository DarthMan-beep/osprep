# Лаб 1 — Постави дозволи на датотека

Напиши ја командата што ги менува дозволите на **`hello_world.py`** така што **само
сопственикот и групата на сопственикот** можат да ја **читаат** и **запишуваат** — без право на извршување за
никого, и без никакви дозволи за останатите.

Датотеката `hello_world.py` веќе постои во тековниот директориум.

## Очекуван резултат

```console
$ ls -l hello_world.py
-rw-rw---- 1 user user 0 ... hello_world.py
```

> **Совет:** со октален `chmod`, секоја цифра претставува класа (сопственик, група, останати) и
> читање = 4, запишување = 2, извршување = 1.
