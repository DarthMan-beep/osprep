# Сортирани уникатни броеви

Напишете **`process.sh`** што обработува датотека со цели броеви користејќи Unix цевка (pipeline).

## Барања

1. Скриптата прифаќа **еден аргумент**: патеката до влезна датотека со еден цел број по линија.
2. Запишува во датотека со име **`result.txt`** (во тековниот директориум) целите броеви
   од влезот **сортирани во опаѓачки нумерички редослед** со **отстранети дупликати**.
3. Ако аргументот недостасува или датотеката не постои, испечатете

   ```
   usage: process.sh <file>
   ```

   на **stderr** и излезете со `1`.
4. Мора да помине `shellcheck`.

## Пример

Влез `nums.txt`:

```
3
1
2
3
2
10
```

По `./process.sh nums.txt`, `result.txt` содржи:

```
10
3
2
1
```

> **Совет:** `sort -rn` сортира нумерички во обратен редослед; `uniq` отстранува **соседни**
> дупликати — поради што доаѓа *по* сортирањето. Комбинирајте ги со `|` цевка
> и пренасочете со `>`.
