# Лаб 1 — Премести и преименувај датотека

Напиши ја командата што ја преместува датотеката **`file123`** од директориумот **`directory1`** во
директориумот **`directory2`**, притоа преименувајќи ја во **`file456`**.

Двата директориума веќе постојат, и `directory1/file123` е присутна.

## Очекуван резултат

```console
$ ls directory1
$ ls directory2
file456
```

> **Совет:** `mv SRC DST` — ако `DST` вклучува ново име, датотеката се преместува *и* се преименува.
