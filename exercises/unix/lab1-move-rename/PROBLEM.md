# Lab 1 — Move and rename a file

Write the command that moves the file **`file123`** from directory **`directory1`** to
directory **`directory2`**, renaming it to **`file456`** in the process.

Both directories already exist, and `directory1/file123` is present.

## Expected result

```console
$ ls directory1
$ ls directory2
file456
```

> **Tip:** `mv SRC DST` — if `DST` includes a new name, the file is moved *and* renamed.
