# Лаб 1 — Броење на недовршени квизови (од stdin)

Читајќи ги податоците за студентите **од стандардниот влез** (еден студент по линија, полиња
`<index> <exam-date> <quiz-status> <points>`), напишете ја командата што го печати
**бројот на студенти што не го завршиле квизот од `14.03.2024`** — т.е. чија
линија ги содржи и `14.03.2024` и статусот `in_progress`.

## Пример

Влез на stdin:
```
21001 14.03.2024 done 80
22002 14.03.2024 in_progress 45
21003 14.03.2024 in_progress 30
22004 16.03.2024 in_progress 50
23005 14.03.2024 done 90
```
Излез:
```
2
```

> **Совет:** поврзете филтри со цевки (pipes) — `grep ... | grep ... | wc -l`.
