# Lab 1 — Count lines, words and characters

Write the command that displays the number of **lines**, **words**, and **characters**
in the file **`OS2.txt`** (which exists in the current directory).

## Expected format

```console
$ <your command>
   3  12  73 OS2.txt
```

(the three numbers are the line, word, and character counts, followed by the filename)

> **Tip:** one command reports all three counts at once.
