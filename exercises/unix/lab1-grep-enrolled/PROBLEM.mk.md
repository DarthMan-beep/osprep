# Лаб 1 — Филтрирај според година на запишување

Датотеката **`OS2.txt`** во тековниот директориум содржи по еден студент по линија:

```
<index> <exam-date> <quiz-status> <points>
```

Индексот започнува со двоцифрената година на запишување (на пр. `21...` значи 2021).

Напиши ја командата што ги печати **само студентите што се запишале во 2021** — т.е. оние чиј
индекс започнува со `21`.

## Пример

За оваа `OS2.txt`:

```
21001 14.03.2024 done 80
22002 15.03.2024 in_progress 45
21003 14.03.2024 done 30
```

излезот е:

```
21001 14.03.2024 done 80
21003 14.03.2024 done 30
```

> **Совет:** `grep` совпаѓа регуларен израз; `^` го прицврстува на почетокот на линијата.
