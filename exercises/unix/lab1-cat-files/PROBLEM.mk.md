# Лаб 1 — Прикажување на две датотеки заедно

Напишете ја командата што ја прикажува содржината на **`OS1.txt`** и **`OS2.txt`**
заедно на екранот — прво целата `OS1.txt`, потоа целата `OS2.txt`.

Двете датотеки постојат во тековниот директориум.

> **Совет:** `cat` може да прифати повеќе од една датотека; ги печати по редослед.
