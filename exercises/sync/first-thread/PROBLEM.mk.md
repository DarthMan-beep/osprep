# Твојата прва нишка

**Нишка (thread)** е посебен тек на извршување што се извршува *напоредно* со твојата
главна програма. Во Java, наједноставниот начин да направиш нишка е да ја наследиш класата
`Thread` и да ја ставиш работата што сакаш да се изврши во нејзиниот метод **`run()`**.

`run()` не го повикуваш сам — наместо тоа, некој повикува **`start()`** на објектот, и Java
го извршува твојот `run()` на сосема нова нишка. Кога треба да се чека да заврши, се
повикува **`join()`**.

## Задача

`Worker` ја наследува `Thread`. Пополни го нејзиниот метод **`run()`** така што ја
пресметува сумата `1 + 2 + … + 100` и ја чува во полето **`result`**.

```java
public class Worker extends Thread {
    public long result = 0;

    @Override
    public void run() {
        // твојот код тука
    }
}
```

Оценувачот го прави следново и очекува `result` да е `5050`:

```java
Worker w = new Worker();
w.start();   // го извршува твојот run() на нова нишка
w.join();    // чека да заврши
// w.result сега мора да е 5050
```

> **Совети:** ова е обичен циклус — `for (int i = 1; i <= 100; i++) sum += i;` — единствената
> разлика е што живее во `run()`, и одговорот го чуваш во `result` (не го `return`-уваш;
> `run()` не враќа ништо).
