# Две нишки работат заедно

Ова е суштината на паралелизмот: **подели ја работата**, изврши ги деловите на одделни
нишки **истовремено**, па **комбинирај** ги резултатите.

Дадена е `RangeWorker` (види *Дадени датотеки*). Ѝ даваш опсег `[from, to]`; нејзиниот
`run()` спие момент, па ја собира таа сума во `result`.

## Задача

Дополни ја **`Launcher.computeWithTwoThreads()`** така што ја собира `1..100` користејќи
**две** нишки:

- едниот работник го обработува `1..50`, другиот `51..100`,
- двете се извршуваат **паралелно**,
- откако двете ќе завршат, врати го **комбинираниот** збир (што е `5050`).

```java
public class Launcher {
    public static long computeWithTwoThreads() throws InterruptedException {
        RangeWorker a = new RangeWorker(1, 50);
        RangeWorker b = new RangeWorker(51, 100);
        // стартувај ги двете, чекај ги двете, па врати a.result + b.result
        return 0;
    }
}
```

> **Совети:** за вистински паралелизам, прво **стартувај ги двете** нишки
> (`a.start(); b.start();`), и дури потоа **чекај ги двете** (`a.join(); b.join();`). Ако
> наместо тоа правиш `a.start(); a.join(); b.start(); b.join();` ќе го добиеш точниот број,
> но ќе се извршат една по друга — стартувај-двете-па-чекај-двете им дозволува да работат
> истовремено. На крај врати `a.result + b.result`.
